Política de Privacidade - Minha Rota GPS Caminhoneiro
1. Informações Coletadas: O aplicativo "Minha Rota" coleta dados de localização precisa em tempo real (GPS) para fornecer navegação e cálculos de rota. Também coletamos as dimensões do caminhão (altura, peso, comprimento) inseridas pelo usuário.
2. Uso dos Dados: A localização é utilizada exclusivamente para o funcionamento do GPS e recálculo de rotas. As dimensões do veículo são usadas para filtrar vias com restrições físicas ou legais (como pontes baixas).
3. Localização em Segundo Plano: Se autorizado, o app pode acessar a localização em segundo plano para manter as instruções de voz ativas enquanto a tela está desligada ou outro app está em uso.
4. Compartilhamento: Não compartilhamos dados de localização ou perfis de veículos com terceiros para fins publicitários. Dados de rota podem ser processados de forma anônima pela nossa parceira de mapas (TomTom).
5. Exclusão de Dados: O usuário pode excluir seus dados de perfil a qualquer momento limpando o cache do aplicativo ou desinstalando-o.
